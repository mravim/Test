Open Session Dialog

     

The open session dialog serves two functions:
  • To open profiling sessions. Double click on an existing session or choose a session and click [Open] to start a profiling session.
  • To edit, copy and delete existing sessions.
The list of available session configurations displays the session name which can be changed when editing a session. In addition, the associated icon to the left of the session name show whether the session is  an "Attach to local JVM" session  an "Attach to profiled JVM (local or remote)" session  a launched application session,  a launched applet session or  a launched Java Web Start session

The facility to open sessions is also embedded in JProfiler's start center.
