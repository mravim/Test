Filters Templates

     

  Filter templates can be saved from the Define Filters tab of the filter settings

A filter template captures all configured filter rules from a session configuration. When saving a filter template, you have to assign a unique name to it. The filter template dialog allows you to reorder, rename and remove existing filter templates.

  The filter template dialog can also be invoked from the session defaults tab of the general settings dialog where you can change the default filter template used for new sessions.
