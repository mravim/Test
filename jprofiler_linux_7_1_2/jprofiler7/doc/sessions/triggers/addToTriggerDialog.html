Method Selection For Triggers

     

Several views in JProfiler display call trees and back traces, such as the call tree, the hot spot view, the allocation call tree and the allocation hot spot view.

In all these views, the context menu shows an  add method trigger action if the currently selected node is a method. That action displays this dialog where you can choose whether to add the method interception to an existing method trigger or whether to create a new method trigger.

If you select "Add to existing method trigger", the list below which displays all existing method triggers is enabled and you have to choose one of them. The select method is added to the selected trigger and the trigger wizard is opened at the "Actions" step, so you can review or modify the existing list of actions.

If you select "Create new method trigger", a new method trigger is created and the trigger wizard is shown at the action step.
