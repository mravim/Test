Profiling Settings Template Dialog

   

  Profiling settings templates can be saved on the profiling settings section of the session settings dialog.

A profiling template contains all profiling settings that can be configured in the profiling settings dialog. When saving a profiling settings template, you have to assign a unique name to it. The profiling settings template dialog allows you to reorder, rename and remove existing profiling settings templates.

  The profiling setting template dialog can also be invoked from the session defaults tab of the general settings dialog where you can change the default profiling settings template used for new sessions.
