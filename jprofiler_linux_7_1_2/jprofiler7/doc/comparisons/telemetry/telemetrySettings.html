VM Telemetry Comparisons View Settings Dialog

   

  The VM telemetry comparison view settings dialog is accessed by bringing any VM telemetry comparison to front and choosing View->View settings from JProfiler's main menu or clicking on the corresponding  toolbar button.
  The following settings are configurable:
  • Scale to fit window
    Determines whether the view operates in the "fixed time scale" or "scale to fit window" mode described in the help on graphs with a time axis.
  • Grid lines for vertical axis
    Controls on what ticks grid lines will be shown along the vertical axis.
  • Symbol for snapshot point
    Controls which symbol is painted for each measurement of a snapshot file. Choose between None, Hollow rectangle and Filled circle (default).
