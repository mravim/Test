Probe Telemetries

     

  The telemetries view is conceptually similar to the VM telemetry views. Each probe can publish one or more telemetries with one or more measurements that are shown in in the same telemetry. The telemetry selector drop-down list at the top offers all available telemetries for the selected probe. Please see the probe overview for more information on the probe-specific telemetries.
  When you move the mouse across a telemetry view, the time at the position of the mouse cursor and the corresponding value on the vertical axis will be shown in JProfiler's status bar. The current value of each data line is always shown next to the corresponding legend entry.

Horizontal and vertical grid lines of the VM telemetry views can be configured in the view settings dialog.

  Please see the help on graphs with a time axis for help on common properties of graph views.
